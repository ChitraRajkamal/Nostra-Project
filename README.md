## Nostra Website
Nostra is a modern, responsive website built using HTML, CSS, and JavaScript. It features a clean user interface, smooth interactions, and an adaptive design suitable for desktops, tablets, and mobile devices.

#Features
Responsive layout with mobile-first design

Smooth navigation and interactive UI elements

Clean and modern visual styling using CSS

JavaScript used for dynamic functionality

Cross-browser compatibility

#Built With
HTML – Markup structure

CSS – Styling and layout

JavaScript – Interactivity and DOM manipulation
